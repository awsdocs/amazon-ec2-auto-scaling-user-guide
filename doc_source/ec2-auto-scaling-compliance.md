+ [Security and Compliance Quick Start Guides](http://aws.amazon.com/quickstart/?awsf.quickstart-homepage-filter=categories%23security-identity-compliance) – These deployment guides discuss architectural considerations and provide steps for deploying baseline environments on AWS that are security and compliance focused\.

## PCI DSS compliance<a name="pci-dss-compliance"></a>

Amazon EC2 Auto Scaling supports the processing, storage, and transmission of credit card data by a merchant or service provider, and has been validated as being compliant with Payment Card Industry \(PCI\) Data Security Standard \(DSS\)\. For more information about PCI DSS, including how to request a copy of the AWS PCI Compliance Package, see [PCI DSS Level 1](https://aws.amazon.com/compliance/pci-dss-level-1-faqs/)\. 

For information on achieving PCI DSS compliance for your AWS workloads, refer to the following compliance guide: 
+ [Payment Card Industry Data Security Standard \(PCI DSS\) 3\.2\.1 on AWS](https://docs.aws.amazon.com/whitepapers/latest/pci-dss-3-2-1-on-aws/pci-dss-3-2-1-on-aws.html)
